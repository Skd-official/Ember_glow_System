## 💡 常见问题

**Q: 重启后还是有错误？**

A: 尝试清除缓存：
```bash
rm -r node_modules/.vite
npm run dev
```

**Q: 为什么我的日期没有触发烟花？**

A: 检查：
1. 当前日期是否真的在节假日内
2. 你的系统是否正确支持 2024-2026 年

**Q: 我想测试 2027 年以后的节假日？**

A: 编辑 `src/utils/lunarCalendar.ts`，在 `lunarHolidaysMap` 中添加新年份的数据。
