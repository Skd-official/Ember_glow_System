## 🔍 故障排除

**Q: 看不到特效？**
A: 
1. 确认已设置 TEST_DATE
2. 刷新页面 (F5)
3. 等待烟花爆炸 3-5 秒后观看
4. 打开 F12 Console 查看是否有错误

**Q: 特效颜色不对？**
A: 
1. 检查 `holidayTextEffects.ts` 中的颜色配置
2. 确认 `effectType` 是否正确传递
3. 看 Console 输出的 effectType 是什么

**Q: 文字太小/太大？**
A: 修改 `canvasRenderer.ts` 第 106 行的字体大小

**Q: 特效运动太快/太慢？**
A: 修改各特效函数中的 `progress * X` 中的 X 值
