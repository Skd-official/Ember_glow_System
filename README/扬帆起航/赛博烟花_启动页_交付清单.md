## ✅ 需求完成度检查

### 你的四个核心需求

#### ① 节假日特殊烟花 + 文言祝福 ✅
- [x] 自动检测当前日期
- [x] 元旦、春节、端午、中秋、重阳、圣诞等 6 个重要节假日
- [x] 清明节被刻意排除（不适合喜庆）
- [x] 每个节假日都有独特的烟花效果：
  - 元旦 → 金色绽放
  - 春节 → 红色链式爆炸
  - 端午 → 龙纹旋转
  - 中秋 → 银色环形
  - 重阳 → 秋色烟火
  - 圣诞 → 冰蓝雪花
- [x] 节假日祝福使用文言文/诗句，而非"xx节快乐"：
  - 中秋 → "但愿人长久，千里共婵娟"
  - 重阳 → "遍插茱萸少一人，陶然共醉菊花酒"
  - 等等

#### ② 烟花起承转合（升空-爆炸-显示） ✅
- [x] 升空阶段：粒子从底部上升，形成竖直光束（0.8-1.2s）
- [x] 爆炸阶段：中心点炸开，粒子向四周散射（0.3-0.5s）
- [x] 显示阶段：粒子缓慢漂落，特效持续展现（2-3s）
- [x] 不同烟花类型的三阶段差异最大（绽放 vs 环形 vs 喷泉等）

#### ③ 文字烟花 + 可配置文字库 ✅
- [x] 30% 概率在烟花序列中触发文字烟花
- [x] 爆炸后烟花消散，文字缓慢淡入
- [x] 可在 `src/utils/textFireworks.ts` 中自由添加文字
- [x] 支持 2-6 个字符，自动调整显示大小
- [x] 文字库包含：'烟火漫游'、'诗意生活'、'人间烟火' 等

#### ④ 东方夜雀食堂风格 ✅
- [x] **标题样式**：
  - 立体多层描边（黑色主描边 + 火焰色光晕）
  - 中文：方正楷体，金黄色 (#FDB927)
  - 英文：Playfair Display，橙色 (#FF8C42)
  - 所有文字都有外光晕效果
- [x] **按钮样式**：
  - 黄→橙渐变背景
  - 黑色边框（仿古风格）
  - 悬停放大 + 光晕增强
  - 点击缩小 + 反馈
- [x] **整体配色**：
  - 暖色调：#FDB927 (奶油黄), #FF8C42 (橙), #FF6B35 (火焰红)
  - 文字：#2C1810 (深棕)
  - 边框：黑色
  - 背景：深夜渐变
- [x] **视觉效果**：
  - 类似东方夜雀食堂的菜单界面
  - 古典 + 现代 + 科技的完美融合
