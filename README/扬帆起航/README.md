# 🎆 烟火漫游启动页系统

## 📌 项目说明

这是一个基于 **React + Canvas + TypeScript** 的高性能烟花系统，用于创建东方美学风格的启动页。

### ✨ 核心特性

- **真实物理引擎** - 模拟真实烟花爆炸和粒子运动
- **多种烟花类型** - 绽放、环形、菊花、柳树、牡丹等8种
- **节日自动检测** - 支持农历和阳历节日识别
- **文字烟花** - 支持自定义文字显示，平时30%、节日80%+显示概率
- **特效配色** - 春节红黄、端午红色、七夕紫红、中秋银蓝等
- **高性能优化** - 保持60fps流畅度，单烟花80-150粒子
